# ai_guided_ultrasound

Developing an AI-guided ultrasound system to assist trainees in real-time scanning and probe positioning, converting 3D medical volumes into ultrasound slices for interactive training simulations, and providing actionable feedback to improve scanning accuracy and efficiency.

## Project Overview

This is a React + TypeScript + Vite-based training platform for AI-guided ultrasound education.

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
